## 컬처맵 기획 개요

### 🎯 서비스 개념
- **컬처맵(CultureMap)**은 사용자가 스스로 계획한 문화 생활 플랜을 기록하고, 일부를 다른 사람과 공유할 수 있는 커뮤니티 기반 문화기록 플랫폼입니다.

### 🧩 핵심 기능 요약
| 기능 | 설명 |
|------|------|
| 문화 플랜 생성 | 사용자가 직접 전시/공연 등 장소를 조합해 플랜을 생성 (날짜 기반) |
| 플랜 공유 | 개인 플랜 중 일부를 게시글로 공유 (Plan → PlanPost 참조 방식) |
| 게시판 열람 | 공유된 플랜들을 전체 공개 게시판에서 확인 가능 |
| 장소 검색 | 한국문화정보원 문화시설 API 기반 장소 검색 기능 |
| 달력 필터링 | 월별 달력 UI에서 날짜 클릭 시 해당 날짜의 플랜 필터링 |
| 회원가입/로그인 | JWT 기반 로그인 및 마이페이지 진입 |

## 기술 스택

### 📦 프로젝트 구조
- 단일 루트 프로젝트 아래 `/frontend` (React) + `/backend` (Spring Boot)

### 💻 백엔드
- **프레임워크**: Spring Boot 3.5.6
- **ORM**: Spring Data JPA + Hibernate
- **인증**: Spring Security + JWT
- **빌드도구**: Gradle
- **DB**:
  - 개발: H2 (인메모리)
  - 운영: MySQL
- **API 문서화**: springdoc-openapi + Swagger UI
- **외부 데이터 연동**: 한국문화정보원 문화시설 API

### 🌐 프론트엔드
- React + TypeScript
- 달력 라이브러리 (예: Litepicker)
- 지도 연동: Kakao 지도 JavaScript SDK (react-kakao-maps-sdk)
- 현위치 기반 지도 중심 설정 및 마커 표시
- 장소 간 경로 시각화 (Polyline 등)

## 도메인 구조 요약

### 👤 Member (회원)
- 회원가입/로그인/JWT 발급
- 마이페이지에서 본인의 플랜 관리

### 📍 Place (문화장소)
- 외부 API 기반으로 확보한 장소 정보
- Plan에 포함될 수 있음 (다대다 관계)

### 🗓️ Plan (사용자 플랜)
- 사용자가 생성하는 문화 일정 (날짜, 장소 리스트 포함)
- 비공개로 저장됨 (기본)

### 📢 PlanPost (플랜 게시글)
- Plan을 참조 (복사 아님)
- 사용자가 “공유하기”를 눌렀을 때만 등록됨
- 제목, 설명, 작성일만 별도 보유


---

## 공유/게시 흐름 요약
1. 사용자 로그인
2. 문화시설 검색 → 플랜 생성
3. 달력/목록에서 나의 플랜 확인
4. 공유하기 → 게시글 작성 화면 → 제목/설명 입력 후 등록
5. PlanPost 게시판에 노출됨


---

## MVP 범위
- ✅ 회원가입/로그인 (JWT)
- ✅ 플랜 생성 (날짜 + 장소 리스트)
- ✅ Plan → 공유 게시글 등록
- ✅ 공유 게시판 열람
- ✅ 문화시설 API 연동 (장소 선택)
- ✅ 달력 UI 기반 필터링

※ 찜, 댓글, 좋아요, 관리자 기능 등은 미포함 (향후 확장 고려)

## 사용자 위치 기반 기능 및 경로 추천 설계

| 항목 | 구현 여부 | 세부 설명 |
|------|------------|-------------|
| 사용자 위치 기반 지도 중심 | ✔️ 가능 | HTML5 Geolocation API로 사용자의 현위치를 받아 Kakao 지도 중심 설정 |
| 주변 장소 마커 표시 | ✔️ 가능 | Kakao Local API로 주변 문화시설 검색 후 마커 표시 |
| 경로 안내 | ✔️ 가능 | Kakao Mobility Directions API로 1~5개 경유지 경로 안내 (자동차 중심). 도보 경로는 Kakao 앱 연동 또는 별도 처리 필요 |
| 최적 방문 순서 추천 | ⭕ 가능 | Kakao API는 자동 정렬 미제공. 프론트/백엔드에서 거리 계산 알고리즘 직접 구현 가능 (예: Nearest Neighbor) |
| 대중교통 경로 | ❌ 불가 | Kakao/네이버 모두 대중교통 경로 API는 미공개. 필요 시 ODsay API 또는 Google Maps Directions API 사용 고려 |
| 무료 사용 가능성 | ✔️ 가능 | 카카오 지도/경로 API는 비상업적 시연 목적일 경우 충분한 무료 쿼터 제공 |
